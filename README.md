# Portal Inforplace - Transformação Digital para o seu Negócio

O **Portal Inforplace** é a vitrine digital definitiva para o ecossistema de soluções da Inforplace. Este projeto foi concebido para ser mais do que um site; é um centro de experiência onde clientes e parceiros podem explorar as ferramentas que estão transformando a gestão empresarial e de engenharia.

## 💼 O Que Você Encontra no Portal

O portal é o ponto central para conhecer nossas soluções líderes de mercado:

- **Sistema Pillar:** Gestão completa e integrada para obras e engenharia.
- **Pillar Mobile:** A potência do Pillar na palma da mão para controle em campo.
- **Sistema Safe:** Excelência na gestão administrativa e financeira.
- **Custo de Obras:** Precisão absoluta no controle imobiliário.
- **NOTAinfo:** Emissão fiscal simplificada e eficiente.
- **PVinfo & Coletor XML:** Agilidade no ponto de venda e gestão inteligente de documentos fiscais.

### 📰 Central de Conteúdo
O portal conta com um **Hub de Postagens dinâmico**, onde compartilhamos atualizações de produtos, notas de lançamento e artigos estratégicos para manter nossos usuários sempre à frente no mercado.

## 🛠️ Excelência Técnica (Stack Tecnológica)

Para entregar uma experiência rápida, segura e moderna, utilizamos o que há de mais avançado no desenvolvimento web:

### Core do Projeto
- **Angular 21:** Implementação de ponta com componentes autônomos e arquitetura *Zoneless* para performance máxima.
- **SSR (Server-Side Rendering):** Conteúdo otimizado para SEO e carregamento instantâneo, garantindo que nossas soluções sejam facilmente encontradas.
- **Dynamic Block CMS:** Um sistema exclusivo de construção de páginas por blocos (Cabeçalhos, Comparação Antes/Depois, Timelines, Vídeos), permitindo que o time de marketing crie conteúdos ricos sem tocar no código.

### Design e Interface
- **Tailwind CSS:** Estilização moderna e responsiva.
- **Lucide Icons:** Interface limpa e intuitiva.

### Infraestrutura e Deploy
- **Docker Ready:** Containerização para deploys consistentes.
- **Node.js 22 LTS:** Base estável e veloz para o motor de renderização.

---

## 🚀 Como Executar o Projeto

1. **Instale as dependências:** `npm install`
2. **Inicie em modo dev:** `npm start` (Acesse `https://localhost:4200`)
3. **Build para produção:** `npm run build`

Desenvolvido com foco na experiência do usuário e inovação constante por **BioOlegari**.
